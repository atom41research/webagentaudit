Introduction

Forge is a modern web application framework designed for developers who value clarity and productivity. It provides a robust set of tools for building database-driven applications with clean, maintainable code.

New to Forge? Start with the Quick Start guide to build your first application in under 10 minutes.

Installation

Forge requires Python 3.10 or higher. Install it using pip: 

pip install forge-framework

Create a new project with the CLI scaffolding tool: 

forge new my-project
cd my-project
forge serve

Your application will be running at http://localhost:8000.

Key Features

Forge ships with everything you need to build production-grade web applications:

  • Expressive Routing Define URL patterns with type-safe parameters
  • Built-in ORM Active Record pattern with migration support
  • Template Engine Jinja2-based templates with layout inheritance
  • Middleware Pipeline Composable request/response processing
  • Authentication System Session and token-based auth out of the box
  • Task Scheduler Cron-like scheduling for background jobs

Configuration

All configuration lives in config/settings.py. Environment-specific overrides are loaded from .env files. Here is a minimal configuration: 

DATABASE_URL = "postgresql://localhost/myapp"
SECRET_KEY = "your-secret-key-here"
DEBUG = True
ALLOWED_HOSTS = ["localhost", "127.0.0.1"]

See the Configuration Reference for the complete list of available settings.
