API Documentation

Welcome to our API docs. Click the sparkle button to ask our AI assistant.

Authentication

All API requests require a Bearer token in the Authorization header. You can generate tokens from your dashboard settings page.

curl -H "Authorization: Bearer YOUR_TOKEN" \
     https://api.example.com/v1/resources

Endpoints

GET /v1/resources

List all resources in your account. Supports pagination via page and per_page query parameters.

POST /v1/resources

Create a new resource. Requires a JSON body with name (string) and type (enum: standard, premium).

GET /v1/resources/:id

Retrieve a single resource by its unique identifier.

DELETE /v1/resources/:id

Delete a resource. This action is irreversible.

Rate Limits

The API enforces rate limits of 100 requests per minute per token. Exceeding this limit returns a 429 Too Many Requests response.
