Messages

Send, retrieve, and manage messages across channels.

Create a Message

Send a message to a specific channel. The message will be delivered to all subscribed members of that channel in real-time.

POST /v3/messages
curl -X POST https://api.nexus.dev/v3/messages \
  -H "Authorization: Bearer sk_live_..." \
  -H "Content-Type: application/json" \
  -d '{
    "channel_id": "ch_abc123",
    "content": "Hello, world!",
    "type": "text"
  }'

Request Body

Parameter Type Description
channel_id required string The ID of the channel to send the message to.
content required string The message content. Supports Markdown formatting.
type string Message type: text, image, or file. Defaults to text.
metadata object Arbitrary key-value pairs attached to the message.

List Messages

Retrieve a paginated list of messages for a given channel. Messages are returned in reverse chronological order.

GET /v3/channels/{channel_id}/messages
curl https://api.nexus.dev/v3/channels/ch_abc123/messages \
  -H "Authorization: Bearer sk_live_..."

Query Parameters

Parameter Type Description
limit integer Number of messages to return (1-100). Default: 25.
before string Cursor for backward pagination. Returns messages before this ID.
after string Cursor for forward pagination. Returns messages after this ID.

Response Format

All message endpoints return a standard envelope with the message object:

{
  "id": "msg_xyz789",
  "channel_id": "ch_abc123",
  "content": "Hello, world!",
  "type": "text",
  "created_at": "2026-03-13T10:30:00Z",
  "author": {
    "id": "usr_def456",
    "name": "Jane Smith"
  }
}
