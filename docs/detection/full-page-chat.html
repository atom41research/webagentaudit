Can you explain how async/await works in Python and show me a practical example?

Python's async/await syntax provides a way to write concurrent code using coroutines. Here's how it works:

Key Concepts:

1. async def defines a coroutine function that can be paused and resumed.

2. await pauses the coroutine until the awaited operation completes, allowing other tasks to run.

3. An event loop manages the execution of coroutines.

import asyncio
import aiohttp

async def fetch_data(url: str) -> dict:
    async with aiohttp.ClientSession() as session:
        async with session.get(url) as response:
            return await response.json()

async def main():
    urls = [
        "https://api.example.com/users",
        "https://api.example.com/posts",
        "https://api.example.com/comments"
    ]
    tasks = [fetch_data(url) for url in urls]
    results = await asyncio.gather(*tasks)
    for result in results:
        print(result)

asyncio.run(main())

In this example, all three HTTP requests run concurrently rather than sequentially. The event loop switches between tasks while each is waiting for a network response, significantly reducing total execution time.
